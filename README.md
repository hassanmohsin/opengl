# Rendering 2D images from 3D objects using OpenGL

## Installation
**Conda**
- `conda env create -f environment.yml`

**Virtual Environment**
- `virtualenv -p $(which python3.6) .venv`
- `source .venv/bin/activate`
- `pip install pygame pyopengl`

## Run
- `source .venv/bin/activate`
- `python -m play.main`
